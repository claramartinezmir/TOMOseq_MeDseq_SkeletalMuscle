# TOMOseq_MeDseq_SkeletalMuscle
Spatial methylomics in skeletal muscle

This repository contains the collection of scripts and python notebooks used to perform the spatial methylomics analysis presented in [C. Martinez Mir et al](https://www.biorxiv.org/content/10.64898/2026.03.16.712025v2). 

The respository is organized in different subfolders:

1. mapping_scripts

Contains all scripts necessary to map MeD-seq data.

2. Regionalization MeDseq

Contains all scripts necessary to perform differential methylation analysis between central and proximal-distal sections.

3. DMR male vs female

Contains all scripts necessary to perform differential methylation analysis between male and female sections.

4. linear model

Contains all scripts necessary to perform the linear regression model with interaction to investigate the contribution of sex and spatial location to DNA methylation.

5. DEG male vs Female

Contains all scripts necessary to perfom differential expression analysis between male and female sections. See for mapping [mouseGastruloids_scRNAseq_tomoseq](https://github.com/anna-alemany/mouseGastruloids_scRNAseq_tomoseq) and [TOMOseq_skeletalMuscle](https://github.com/claramartinezmir/TOMOseq_skeletalMuscle) for pre-processing of spatial transcriptomics data.
